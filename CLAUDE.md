# Global CLAUDE.md

## Communication

- Respond in English
- When unsure about implementation details, ask before proceeding
- Ask before adding new dependencies/packages

## Git Workflow

**Branch naming**: `<type>/<issue-number>-<slug>`
- Types: `feature/`, `fix/`, `docs/`, `refactor/`

**PR description**: Always include `Resolves #<issue-number>` (extracted from branch name) to auto-close the issue.

**Commits**: Conventional commits (`feat:`, `fix:`, `refactor:`, `docs:`, `chore:`)
